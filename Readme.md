# 🍋 LIME Text NLP Explainer Dashboard

An interactive dashboard for explaining NLP model predictions using LIME (Local Interpretable Model-agnostic Explanations).
## Features
- 🔍 Explain predictions of any text classification model
- 🎯 Highlight important words with color-coded visualization
- 📊 Multiple model support (BERT, DistilBERT, SVM, etc.)
- 📈 Interactive charts and statistics
- 💾 Save and export explanations
